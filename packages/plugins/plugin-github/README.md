# @dependabit/plugin-github

GitHub API file/release access plugin for dependency tracking.

## Overview

This plugin provides GitHub-specific dependency tracking, including file changes and release monitoring.

## Features

- GitHub file tracking
- Release monitoring
- Tag and branch tracking

## Installation

```bash
pnpm add @dependabit/plugin-github
```

## Usage

```typescript
import { GitHubPlugin } from '@dependabit/plugin-github';
```

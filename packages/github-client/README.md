# @dependabit/github-client

GitHub API wrapper with rate limiting and authentication.

## Overview

This package provides a robust wrapper around the GitHub API with built-in rate limiting, authentication handling, and common operations.

## Features

- Octokit wrapper with rate limit handling
- Multiple authentication strategies
- Commit and release fetching
- Issue creation and management

## Installation

```bash
pnpm add @dependabit/github-client
```

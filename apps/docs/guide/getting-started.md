# Introduction

Dependabit automatically discovers, tracks, and monitors external dependencies referenced in your codebase using LLM-powered analysis. Unlike traditional dependency managers that only track package manifests, Dependabit finds informational dependencies like GitHub repos, documentation sites, API references, research papers, and more.

> **Pre-1.0 software** — APIs are subject to change between minor versions. Pin to exact versions in production.

## Why Dependabit?

Modern projects reference many external resources that package managers don't track: blog posts, specifications, research papers, upstream GitHub repos, and documentation sites. When those resources change or disappear, your project quietly rots. Dependabit closes that gap.

# Functions

## Manifest

### `updateDependency`
Updates a single dependency entry in the on-disk manifest by ID.
```ts
updateDependency(path: string, dependencyId: string, updates: Partial<DependencyEntry>): Promise<{ version: "1.0.0"; generatedAt: string; generatedBy: { action: string; version: string; llmProvider: string; llmModel?: string }; repository: { owner: string; name: string; branch: string; commit: string }; dependencies: { id: string; url: string; type: "reference-implementation" | "schema" | "documentation" | "research-paper" | "api-example" | "other"; accessMethod: "context7" | "arxiv" | "openapi" | "github-api" | "http"; name: string; currentStateHash: string; detectionMethod: "llm-analysis" | "manual" | "package-json" | "requirements-txt" | "code-comment"; detectionConfidence: number; detectedAt: string; lastChecked: string; referencedIn: { file: string; line?: number; context?: string }[]; changeHistory: { detectedAt: string; severity: "breaking" | "major" | "minor"; falsePositive: boolean; oldVersion?: string; newVersion?: string; issueNumber?: number }[]; description?: string; currentVersion?: string; lastChanged?: string; auth?: { type: "token" | "oauth" | "basic" | "none"; secretEnvVar?: string }; monitoring?: { enabled: boolean; checkFrequency: "hourly" | "daily" | "weekly" | "monthly"; ignoreChanges: boolean; severityOverride?: "breaking" | "major" | "minor" } }[]; statistics: { totalDependencies: number; byType: Record<string, number>; byAccessMethod: Record<string, number>; byDetectionMethod: Record<string, number>; averageConfidence: number; falsePositiveRate?: number } }>
```
**Parameters:**
- `path: string` — Path to the manifest file.
- `dependencyId: string` — UUID of the dependency to update.
- `updates: Partial<DependencyEntry>` — Partial `DependencyEntry` fields to merge.
**Returns:** `Promise<{ version: "1.0.0"; generatedAt: string; generatedBy: { action: string; version: string; llmProvider: string; llmModel?: string }; repository: { owner: string; name: string; branch: string; commit: string }; dependencies: { id: string; url: string; type: "reference-implementation" | "schema" | "documentation" | "research-paper" | "api-example" | "other"; accessMethod: "context7" | "arxiv" | "openapi" | "github-api" | "http"; name: string; currentStateHash: string; detectionMethod: "llm-analysis" | "manual" | "package-json" | "requirements-txt" | "code-comment"; detectionConfidence: number; detectedAt: string; lastChecked: string; referencedIn: { file: string; line?: number; context?: string }[]; changeHistory: { detectedAt: string; severity: "breaking" | "major" | "minor"; falsePositive: boolean; oldVersion?: string; newVersion?: string; issueNumber?: number }[]; description?: string; currentVersion?: string; lastChanged?: string; auth?: { type: "token" | "oauth" | "basic" | "none"; secretEnvVar?: string }; monitoring?: { enabled: boolean; checkFrequency: "hourly" | "daily" | "weekly" | "monthly"; ignoreChanges: boolean; severityOverride?: "breaking" | "major" | "minor" } }[]; statistics: { totalDependencies: number; byType: Record<string, number>; byAccessMethod: Record<string, number>; byDetectionMethod: Record<string, number>; averageConfidence: number; falsePositiveRate?: number } }>` — The updated manifest.
**Throws:** If the merged entry fails schema validation.

# Manifest

| Class | Description |
|-------|-------------|
| [readManifest](readmanifest.md) | Reads a manifest JSON file from disk and validates it against
`DependencyManifestSchema`. |
| [writeManifest](writemanifest.md) | Serialises a manifest to pretty-printed JSON and writes it to disk. |
| [updateDependency](updatedependency.md) | Updates a single dependency entry in the on-disk manifest by ID. |
| [addDependency](adddependency.md) | Appends a new dependency entry to the on-disk manifest. |
| [removeDependency](removedependency.md) | Removes a dependency from the on-disk manifest by ID. |
| [mergeManifests](mergemanifests.md) | Merges an `updated` manifest generated by the detector over an `existing`
one, optionally preserving manual entries and accumulated change history. |
| [createEmptyManifest](createemptymanifest.md) | Creates a minimal, valid manifest with an empty `dependencies` array. |
| [readConfig](readconfig.md) | Reads a YAML configuration file from disk, parses it, and validates it
against `DependabitConfigSchema`. |
| [parseConfig](parseconfig.md) | Parses a YAML string into a validated `DependabitConfig`. |
| [stringifyConfig](stringifyconfig.md) | Serialises a validated `DependabitConfig` to a YAML string. |
| [getEffectiveMonitoringRules](geteffectivemonitoringrules.md) | Resolves the effective monitoring rules for a specific dependency URL by
merging global config defaults with any per-URL override defined in
`config.dependencies`. |
| [shouldIgnoreUrl](shouldignoreurl.md) | Returns `true` if the given URL matches any exclusion rule defined in
`config.ignore` (exact URL list or regex pattern list). |
